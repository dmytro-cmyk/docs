## Base URL

All API v1 endpoints use the following base URL:

```
https://app.zenedu.com/api/v1
```

## Authentication

API v1 uses Bearer token authentication via Laravel Sanctum. Include your token in every request:

```bash
curl -X GET https://app.zenedu.com/api/v1/bots \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Accept: application/json"
```

<Warning>
  The `Accept: application/json` header is required for all API v1 requests.
</Warning>

Generate an API token in your Zenedu dashboard under account settings. The token is shown only once -- store it securely.

## Pro endpoints

Some endpoints require an active Pro subscription. These are marked with **(Pro)** in the documentation. Calling a Pro endpoint without an active subscription returns `403 Forbidden`.

## Pagination

List endpoints return paginated responses with `data`, `links`, and `meta` fields. Use the `per_page` query parameter to control page size (default: 30).
