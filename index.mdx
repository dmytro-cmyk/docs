## Overview

Zenedu provides two APIs for integrating with the platform:

<Columns>
  <Card title="API v1" icon="code" href="/api-reference/introduction">
    Server-to-server API for managing bots, subscribers, orders, products, and offers. Uses Sanctum Bearer token authentication.
  </Card>
  <Card title="Mini App API" icon="mobile" href="/mini-app/overview">
    API for Telegram Mini App frontend. Uses JWT authentication with Telegram initData validation.
  </Card>
</Columns>

## Quick start

<Steps>
  <Step title="Get your API token">
    Generate an API token in your Zenedu dashboard under account settings.
  </Step>
  <Step title="Make your first request">
    ```bash
    curl -X GET https://app.zenedu.com/api/v1/me \
      -H "Authorization: Bearer YOUR_API_TOKEN" \
      -H "Accept: application/json"
    ```
  </Step>
  <Step title="Explore endpoints">
    Browse the [API v1 reference](/api-reference/introduction) to see all available endpoints.
  </Step>
</Steps>

## Base URL

All API v1 requests should be made to:

```
https://app.zenedu.com/api/v1
```

Mini App API requests use:

```
https://app.zenedu.com/api/mini-apps
```
